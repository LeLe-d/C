阶段一：基础入门 - 运算符

运算符是 C 语言中用于执行各种操作（如计算、比较、逻辑判断等）的特殊符号。运算符作用于一个或多个操作数 (Operand)（通常是变量或常量）。理解不同的运算符及其优先级是编写正确、高效代码的关键。

1. 算术运算符 (Arithmetic Operators)

用于执行基本的数学运算。

运算符 名称 示例 说明
+ 加法 a + b 计算 a 与 b 的和
- 减法 a - b 计算 a 与 b 的差
* 乘法 a * b 计算 a 与 b 的积
/ 除法 a / b 计算 a 除以 b 的商。注意： 若 a 和 b 均为整数，则结果为整数（舍弃小数部分）；若任一为浮点数，则结果为浮点数。
% 取模 (取余) a % b 计算 a 除以 b 的余数。操作数必须是整数类型。
++ 自增 a++ (后置) 或 ++a (前置) 将变量的值加 1。前置是先加1再使用变量，后置是先使用变量再加1。
-- 自减 a-- (后置) 或 --a (前置) 将变量的值减 1。规则同自增。 
int a = 10, b = 3;
int sum = a + b;   // sum is 13
int diff = a - b;  // diff is 7
int product = a * b; // product is 30
int quotient_int = a / b; // quotient_int is 3 (整数除法)
float quotient_float = (float)a / b; // quotient_float is 3.333...
int remainder = a % b;  // remainder is 1

int c = 5;
int d = ++c; // c 变为 6, d 被赋值为 6 (前置)
int e = 5;
int f = e++; // f 被赋值为 5, e 变为 6 (后置)

printf("Sum: %d, Quotient (int): %d, Remainder: %d\n", sum, quotient_int, remainder);
printf("Quotient (float): %.2f\n", quotient_float);
printf("c=%d, d=%d, e=%d, f=%d\n", c, d, e, f);

2. 关系运算符 (Relational Operators)

用于比较两个操作数之间的关系，结果为真 (1) 或 假 (0) (在 C 语言中用整数表示)。常用于条件语句中。

运算符 名称 示例 说明
== 等于 a == b 判断 a 是否等于 b
!= 不等于 a != b 判断 a 是否不等于 b
> 大于 a > b 判断 a 是否大于 b
< 小于 a < b 判断 a 是否小于 b
>= 大于等于 a >= b 判断 a 是否大于或等于 b
<= 小于等于 a <= b 判断 a 是否小于或等于 b 
int x = 5, y = 10;
printf("x == y: %d\n", x == y);   // 输出 0 (假)
printf("x != y: %d\n", x != y);   // 输出 1 (真)
printf("x < y: %d\n", x < y);    // 输出 1 (真)
printf("x >= 5: %d\n", x >= 5);   // 输出 1 (真)

3. 逻辑运算符 (Logical Operators)

用于组合或否定逻辑表达式（通常是关系运算的结果），结果也为真 (1) 或假 (0)。

运算符 名称 示例 说明
&& 逻辑与 (AND) expr1 && expr2 当 expr1 和 expr2 都为真 (非零) 时，结果为真 (1)；否则为假 (0)。
短路求值： 如果 expr1 为假 (0)，则不再计算 expr2。
|| 逻辑或 (OR) expr1 || expr2 当 expr1 或 expr2 中至少有一个为真 (非零) 时，结果为真 (1)；否则为假 (0)。
短路求值： 如果 expr1 为真 (非零)，则不再计算 expr2。
! 逻辑非 (NOT) !expr 如果 expr 为假 (0)，结果为真 (1)；如果 expr 为真 (非零)，结果为假 (0)。 
int age = 25;
int has_ticket = 1; // 1 表示真 (有票)
int is_vip = 0;    // 0 表示假 (不是 VIP)

if (age >= 18 && has_ticket) {
    printf("允许入场。\n");
}

if (is_vip || age > 65) {
    printf("享受优先通道。\n");
} else {
     printf("普通通道。\n");
}

if (!is_vip) {
    printf("不是 VIP 用户。\n");
}

4. 位运算符 (Bitwise Operators)

直接对操作数的二进制位进行操作。操作数通常是整型。常用于底层编程、硬件控制、数据压缩和加密等。

运算符 名称 示例 说明
& 按位与 (AND) a & b 对 a 和 b 的每一位进行与操作 (都为 1 则结果为 1，否则为 0)。
| 按位或 (OR) a | b 对 a 和 b 的每一位进行或操作 (任一为 1 则结果为 1，否则为 0)。
^ 按位异或 (XOR) a ^ b 对 a 和 b 的每一位进行异或操作 (位相异则结果为 1，相同则为 0)。
~ 按位取反 (NOT) ~a 对 a 的每一位取反 (0 变 1，1 变 0)。这是一个一元运算符。
<< 左移 a << n 将 a 的所有位向左移动 n 位，右侧空出的位补 0。效果通常等同于 a 乘以 2n。
>> 右移 a >> n 将 a 的所有位向右移动 n 位。对于 unsigned 类型或正数 signed 类型，左侧空出的位补 0 (逻辑右移)；对于负数 signed 类型，结果是实现定义的 (可能是补 0 或补符号位，即算术右移)。效果通常等同于 a 除以 2n。 
unsigned char p = 5;  // 二进制: 00000101
unsigned char q = 12; // 二进制: 00001100
unsigned char result_and = p & q; // 结果: 00000100 (十进制 4)
unsigned char result_or = p | q;  // 结果: 00001101 (十进制 13)
unsigned char result_xor = p ^ q;  // 结果: 00001001 (十进制 9)
unsigned char result_not = ~p;   // 结果: 11111010 (取决于 char 是 signed 还是 unsigned 及位数，此处假设 8 位 unsigned)
unsigned char result_left_shift = p << 2; // 结果: 00010100 (十进制 20)
unsigned char result_right_shift = q >> 1; // 结果: 00000110 (十进制 6)

printf("p & q = %d\n", result_and);
printf("p << 2 = %d\n", result_left_shift);

5. 赋值运算符 (Assignment Operators)

用于给变量赋值。

运算符 示例 等价于 说明
= a = b - 简单赋值：将 b 的值赋给 a。
+= a += b a = a + b 加后赋值
-= a -= b a = a - b 减后赋值
*= a *= b a = a * b 乘后赋值
/= a /= b a = a / b 除后赋值
%= a %= b a = a % b 取模后赋值
&= a &= b a = a & b 按位与后赋值
|= a |= b a = a | b 按位或后赋值
^= a ^= b a = a ^ b 按位异或后赋值
<<= a <<= n a = a << n 左移后赋值
>>= a >>= n a = a >> n 右移后赋值 
int score = 100;
score += 10; // score 现在是 110
score -= 20; // score 现在是 90
printf("Score: %d\n", score);

int flags = 5; // 0101
flags |= 2; // flags |= 0010 => 0111 (十进制 7)
printf("Flags: %d\n", flags);

6. 杂项运算符 (Miscellaneous Operators)

包括一些不属于上述类别但很重要的运算符。

运算符 名称 示例 说明
sizeof 尺寸运算符 sizeof(type) 或 sizeof(variable) 返回操作数（类型或变量）所占用的内存字节数。结果是 size_t 类型 (通常是无符号整数)。
? : 条件运算符 (三元运算符) condition ? value_if_true : value_if_false 如果 condition 为真 (非零)，则表达式的值为 value_if_true；否则为 value_if_false。是 if-else 语句的简洁形式。
, 逗号运算符 expr1, expr2 顺序计算 expr1 和 expr2，整个表达式的值是 expr2 的值。常用于 for 循环中同时更新多个变量。
& 地址运算符 &variable 返回变量 variable 的内存地址。与指针紧密相关。
* 间接运算符 (解引用) *pointer 访问指针 pointer 所指向地址处的值。与指针紧密相关。
(type) 类型转换 (float)int_var 将表达式的值强制转换为指定的类型。
. 和 -> 成员访问 struct_var.member
struct_ptr->member		 用于访问结构体 (struct) 或联合体 (union) 的成员。. 用于普通变量，-> 用于指向结构/联合体的指针。
[] 数组下标 array[index] 用于访问数组中的元素。
() 函数调用 function_name(args) 用于调用函数。 
int num = 10;
printf("Size of int: %zu bytes\n", sizeof(int));   // 输出 int 类型的字节数
printf("Size of num: %zu bytes\n", sizeof(num));    // 输出变量 num 的字节数

int a = 5, b = 8;
int max_val = (a > b) ? a : b; // max_val 会被赋值为 8
printf("Max value: %d\n", max_val);

int i, j;
for (i = 0, j = 5; i < 3; i++, j--) { // 逗号运算符用于初始化和更新
    printf("i=%d, j=%d\n", i, j);
}

int val = 100;
int* ptr = &val; // & 获取地址
printf("Value via pointer: %d\n", *ptr); // * 解引用指针

float result = (float)10 / 3; // 类型转换
printf("Result: %.2f\n", result);
运算符优先级和结合性： 不同的运算符有不同的执行优先级（例如乘除优先于加减）和结合性（例如赋值运算符是右结合 a=b=c 等价于 a=(b=c)）。当不确定优先级时，使用圆括号 () 来明确指定计算顺序，这也能提高代码的可读性。
