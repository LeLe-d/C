阶段一：基础入门 - 第一个程序：Hello, World!

在搭建好开发环境后，我们来编写并运行第一个 C 语言程序。这几乎是所有编程语言入门的传统——打印 "Hello, World!" 到屏幕上。虽然简单，但它包含了 C 程序的基本骨架。

1. 编写代码

打开你选择的 IDE (如 VS Code, Dev-C++) 或一个纯文本编辑器 (如 Notepad++, Gedit)，创建一个新文件，并输入以下代码。将文件保存为 hello.c (文件名可以自定，但后缀必须是 .c)。

#include <stdio.h> // 包含标准输入输出头文件

int main() { // 程序的主入口函数
    // 调用 printf 函数打印字符串到控制台
    printf("Hello, World!\n");

    return 0; // 向操作系统报告程序成功结束
}

2. 理解基本结构

让我们逐行分解这个简单的程序：

  • #include <stdio.h>
    这是一个预处理指令。它告诉编译器在实际编译之前，将 "stdio.h" 这个头文件的内容包含进来。stdio.h 是 C 标准库的一部分，提供了标准输入 (Standard Input) 和标准输出 (Standard Output) 相关的功能，我们后面要用的 printf 函数就在这个文件里声明的。
  • int main()
    这是程序的主函数，也是 C 程序的入口点。当你的程序运行时，操作系统会首先调用 main 函数。
    • int：表示 main 函数执行完毕后会返回一个整数值给操作系统。
    • main：是 C 语言规定的主函数名，必须要有。
    • ()：表示这是一个函数。括号内可以包含传递给函数的参数，这里为空，表示不接收任何参数。
  • { 和 }
    这对花括号定义了一个代码块，main 函数的所有代码都必须写在这对花括号之间。
  • printf("Hello, World!\n");
    这是一个函数调用语句。
    • printf：是定义在 stdio.h 中的一个函数，用于将信息打印（输出）到控制台（通常是你的电脑屏幕）。
    • ("Hello, World!\n")：是传递给 printf 函数的参数，是一个字符串字面量。双引号内的内容就是要打印的文本。
    • \n：是一个转义字符，表示一个换行符。它会让光标移动到下一行的开头，这样后续的输出就不会紧跟在 "Hello, World!" 后面了。
    • ;：分号是 C 语言中语句的结束符，每个独立的语句后面都必须跟一个分号。
  • return 0;
    这个语句结束 main 函数的执行，并返回一个值。按照惯例，return 0; 表示程序正常结束。如果返回非零值，通常表示程序遇到了某种错误。这个返回值是给调用该程序的操作系统或父进程看的。

3. 编译和运行

编写好代码后，需要经过两个步骤才能看到结果：编译和运行。

方法一：使用命令行 (以 GCC 为例)

  1. 打开终端或命令提示符:
    • Windows: 打开 CMD 或 PowerShell。
    • macOS/Linux: 打开终端 (Terminal)。
  2. 导航到文件目录: 使用 cd 命令切换到你保存 hello.c 文件的那个目录。例如： cd Desktop 或 cd C:\Users\YourName\Documents\C_Projects。
  3. 编译 (Compile): 在命令行中输入以下命令并按回车：
    gcc hello.c -o hello
    • gcc: 调用 GCC 编译器。
    • hello.c: 你要编译的源文件名。
    • -o hello: (字母 o，不是数字 0) 指定输出的可执行文件的名字为 hello (Windows 下会自动生成 hello.exe)。如果省略 -o hello，Linux/macOS 下默认生成 a.out，Windows (MinGW) 下默认生成 a.exe。
    如果没有错误信息输出，说明编译成功，当前目录下会生成一个名为 hello (或 hello.exe) 的文件。
  4. 运行 (Run):
    • Linux/macOS: 在命令行输入 ./hello 并回车。 (./ 表示运行当前目录下的文件)
    • Windows: 在命令行输入 hello 或 .\hello (取决于你的环境配置) 并回车。

执行后，你应该能在命令行窗口看到输出：

Hello, World!

方法二：使用 IDE

如果你使用的是 IDE (如 VS Code 配置好 C/C++ 扩展, Dev-C++, CLion)，操作通常会更简单：

  1. 打开或创建 hello.c 文件。
  2. 找到 IDE 中的 "编译" (Compile/Build) 按钮或菜单项，点击它。IDE 会自动调用编译器。
  3. 找到 IDE 中的 "运行" (Run/Execute) 按钮或菜单项，点击它。IDE 会执行编译好的程序，并通常会在内置的控制台或终端窗口显示输出 "Hello, World!"。

关键理解： 无论使用哪种方法，核心流程都是：编写源代码 -> 编译器编译 -> 生成可执行文件 -> 运行可执行文件 -> 查看输出。IDE 只是将这些步骤自动化和集成了。

恭喜！你已经成功编写并运行了你的第一个 C 程序。现在你对 C 程序的基本结构（包含头文件、主函数、语句、返回）有了初步的认识。
