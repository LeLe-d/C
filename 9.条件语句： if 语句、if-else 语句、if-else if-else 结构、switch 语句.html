阶段二：流程控制 - C语言条件语句

条件语句允许程序根据不同的条件执行不同的代码路径。它们是控制程序流程的基础，使得程序能够做出决策。C 语言主要提供以下几种条件语句：if、if-else、if-else if-else 结构和 switch。

1. if 语句

if 语句是最基本的条件语句。它检查一个条件是否为真（非零），如果为真，则执行紧跟其后的代码块。

基本语法：

if (condition) {
    // 如果 condition 为真 (非零)，执行这里的语句
}

condition 是一个表达式，其结果会被评估为真（非零）或假（零）。如果 condition 为真，花括号 {} 内的代码块将被执行。如果只有一个语句需要执行，花括号可以省略，但不推荐这样做，以避免混淆。

示例代码：

#include 

int main() {
    int score = 75;

    if (score >= 60) {
        printf("恭喜，你及格了！\n");
    }

    printf("程序继续执行...\n");
    return 0;
}

在这个例子中，因为 score (75) 大于等于 60，条件为真，所以会打印 "恭喜，你及格了！"。

2. if-else 语句

if-else 语句提供了一个在条件为假时执行的备选代码路径。

基本语法：

if (condition) {
    // 如果 condition 为真 (非零)，执行这里的语句
} else {
    // 如果 condition 为假 (零)，执行这里的语句
}

如果 condition 为真，执行 if 后面的代码块；如果 condition 为假，执行 else 后面的代码块。

示例代码：

#include 

int main() {
    int number = 10;

    if (number % 2 == 0) {
        printf("数字 %d 是偶数。\n", number);
    } else {
        printf("数字 %d 是奇数。\n", number);
    }
    return 0;
}

这里，number % 2 == 0 判断 number 是否能被 2 整除。因为 10 是偶数，条件为真，执行 if 块。

3. if-else if-else 结构

当需要检查多个条件时，可以使用 if-else if-else 结构（也称为 "else-if 梯子"）。它按顺序检查一系列条件，一旦找到一个为真的条件，就执行对应的代码块，然后跳过其余的 else if 和 else 部分。

基本语法：

if (condition1) {
    // 如果 condition1 为真，执行这里
} else if (condition2) {
    // 如果 condition1 为假 且 condition2 为真，执行这里
} else if (condition3) {
    // 如果 前面的条件都为假 且 condition3 为真，执行这里
}
// ... 可以有更多 else if
else {
    // 如果 以上所有条件都为假，执行这里 (可选的 'else' 部分)
}

最后的 else 部分是可选的，用于处理所有前面条件都不满足的情况。

示例代码：

#include 

int main() {
    int score = 85;
    char grade;

    if (score >= 90) {
        grade = 'A';
    } else if (score >= 80) {
        grade = 'B'; // score 为 85，这个条件为真，执行这里
    } else if (score >= 70) {
        grade = 'C';
    } else if (score >= 60) {
        grade = 'D';
    } else {
        grade = 'F';
    }

    printf("分数 %d 对应的等级是：%c\n", score, grade);
    return 0;
}

因为 score 是 85，第一个条件 score >= 90 为假，检查第二个条件 score >= 80 为真，所以 grade 被赋值为 'B'，然后整个 if-else if-else 结构结束。

4. switch 语句

switch 语句是一种特殊的多路分支结构，它根据一个整型表达式（包括字符类型 char）的值，将其与一系列 case 标签后的常量值进行比较。当找到匹配的 case 时，程序跳转到该 case 后的语句开始执行。

基本语法：

switch (expression) { // expression 必须是整型或字符型
    case constant1:
        // 如果 expression 的值等于 constant1，执行这里的语句
        break; // 跳出 switch 语句
    case constant2:
        // 如果 expression 的值等于 constant2，执行这里的语句
        break;
    // ... 可以有更多 case
    default: // 可选
        // 如果 expression 的值不匹配任何 case，执行这里的语句
        break; // default 里的 break 通常也是好习惯
}
  • expression：其值将被用于比较。必须是整数类型（int, char, enum 等）。
  • case constantX：constantX 必须是常量表达式（字面量或 const 变量）。每个 case 的常量值必须唯一。
  • break：非常重要！ break 语句用于在执行完匹配的 case 块后退出 switch 结构。如果没有 break，程序会继续执行下一个 case 的代码块（称为“贯穿”或 "fall-through"），这通常不是期望的行为。
  • default：可选的。如果 expression 的值与所有 case 都不匹配，则执行 default 块中的代码。

示例代码：

#include 

int main() {
    char grade = 'B';

    printf("你的等级是 %c，评价：", grade);

    switch (grade) {
        case 'A':
            printf("优秀！\n");
            break;
        case 'B':
            printf("良好！\n"); // grade 是 'B', 匹配这个 case
            break; // 执行完后跳出 switch
        case 'C':
            printf("中等。\n");
            break;
        case 'D':
            printf("及格。\n");
            break;
        case 'F':
            printf("不及格。\n");
            break;
        default:
            printf("无效的等级。\n");
            break;
    }
    return 0;
}

因为 grade 是 'B'，程序会执行 case 'B': 后面的 printf 语句，然后遇到 break，跳出 switch 结构。

注意：switch 语句通常比冗长的 if-else if-else 结构在处理基于单个变量值的多路选择时更清晰、有时也更高效。但它仅限于整型（或可提升为整型的类型）和常量比较。

总结

条件语句是 C 语言编程的核心部分，它们使得程序能够根据不同的输入或状态做出反应。

  • 使用 if 进行简单的单条件检查。
  • 使用 if-else 提供两种执行路径。
  • 使用 if-else if-else 处理多个互斥的条件。
  • 当基于一个整型变量的多个特定常量值进行分支时，优先考虑使用 switch，并务必记得使用 break 防止意外的“贯穿”。

合理地选择和使用这些条件语句，可以编写出逻辑清晰、功能强大的程序。
