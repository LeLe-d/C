阶段一：基础入门 - 数据类型

在 C 语言中，数据类型用于指定变量可以存储的数据种类以及所需的内存大小。声明变量时必须指定其数据类型。这有助于编译器理解如何存储数据、可以执行哪些操作以及如何解释内存中的位模式。

C 语言的基本数据类型主要可以分为以下几类：

1. 整型 (Integer Types)

用于存储整数（没有小数部分的数字）。根据存储范围和内存占用的不同，有多种整型类型。

类型关键字 描述 典型内存大小 (可能变化) 典型取值范围 (可能变化)
short (或 short int) 短整型 2 字节 -32,768 到 32,767
int 基本整型 (最常用) 通常 4 字节 (有时 2 或 8) 约 -21亿 到 +21亿 (4字节时)
long (或 long int) 长整型 通常 4 或 8 字节 范围比 int 大或相等
long long (或 long long int) (C99 标准) 更长的整型 通常 8 字节 范围非常大

修饰符：

  • signed (默认): 可以存储正数、负数和零。
  • unsigned: 只能存储非负数（零和正数）。使用 unsigned 可以将正数的存储范围扩大约一倍。例如 unsigned int。
// 声明和初始化整型变量
int age = 30;
short count = 100;
long population = 7800000000L; // L 后缀表示 long 类型常量
long long very_large_number = 123456789012345LL; // LL 后缀表示 long long
unsigned int positive_value = 4000000000U; // U 后缀表示 unsigned

printf("年龄: %d\n", age); // %d 用于输出 signed int
printf("人口: %ld\n", population); // %ld 用于输出 long int
printf("无符号值: %u\n", positive_value); // %u 用于输出 unsigned int

2. 浮点型 (Floating-Point Types)

用于存储带有小数部分的数字，也称为实数。

类型关键字 描述 典型内存大小 (可能变化) 精度 (有效数字位数)
float 单精度浮点型 4 字节 约 6-7 位
double 双精度浮点型 (更常用，精度更高) 8 字节 约 15-16 位
long double 扩展精度浮点型 通常 10, 12 或 16 字节 更高精度 (依赖于实现)
注意： 浮点数在计算机中存储的是近似值，进行比较时要小心精度问题。直接用 == 比较两个浮点数是否相等通常是不可靠的。 
// 声明和初始化浮点型变量
float price = 19.99f; // f 或 F 后缀表示 float 类型常量
double pi = 3.141592653589793; // 默认的小数常量是 double 类型
long double very_precise_value = 0.1234567890123456789L; // L 后缀 (大写) 表示 long double

printf("价格: %.2f\n", price); // %f 用于输出 float 或 double, .2 指定保留两位小数
printf("圆周率: %lf\n", pi);  // %lf 在 printf 中通常与 %f 效果相同，但在 scanf 中必须用 %lf 读取 double
printf("高精度值: %Lf\n", very_precise_value); // %Lf 用于输出 long double

3. 字符型 (Character Type)

用于存储单个字符（如字母 'A'、数字 '5'、符号 '$'）。

类型关键字 描述 内存大小 存储内容
char 字符型 1 字节 存储字符的 ASCII 值 (或其他字符集编码值)

实际上，char 类型在内部存储的是一个小的整数值（通常是字符对应的 ASCII 码）。它可以像整数一样参与运算。字符常量用单引号 ' ' 括起来。 

// 声明和初始化字符型变量
char grade = 'A';
char initial = 'J';
char digit = '7';
char symbol = '$';
char newline_char = '\n'; // 转义字符表示换行

printf("等级: %c\n", grade);      // %c 用于输出字符
printf("等级的ASCII码: %d\n", grade); // 可以用 %d 输出字符对应的整数值

4. void 类型

void 是一个特殊的类型，表示“无类型”或“空类型”。它主要用于以下三个方面：

  • 函数返回类型： 当函数不返回任何值时，其返回类型应声明为 void。 
    void print_message(char* msg) {
    printf("%s\n", msg);
    // 没有 return 语句返回值
}
  • 函数参数列表： 当函数不接受任何参数时，可以在括号内写 void (虽然在现代 C 中，更常见的做法是直接写空括号 ())。 
    int get_random_number(void) {
    // ... 函数体 ...
    return 42; // 示例
}
// 等价于： int get_random_number(); (在函数原型中) 或 int get_random_number() { ... } (在定义中)
  • 通用指针 (void*)： void* 是一种特殊的指针类型，可以指向任何类型的数据。它提供了一种泛型编程的机制，但在使用前通常需要将其强制转换为具体的类型指针。 (这是更高级的用法) 
    int num = 10;
void* generic_ptr = &num; // 指向一个 int

// 需要转换才能解引用
int* int_ptr = (int*)generic_ptr;
printf("Value via void*: %d\n", *int_ptr);
sizeof 运算符： 你可以使用 sizeof 运算符来获取某个数据类型或变量在当前系统上占用的内存字节数。例如：sizeof(int) 会返回 int 类型占用的字节数。这对于理解不同平台上的差异很有帮助。 
printf("Size of int: %zu bytes\n", sizeof(int));
printf("Size of float: %zu bytes\n", sizeof(float));
printf("Size of char: %zu bytes\n", sizeof(char));
// %zu 是用于打印 sizeof 返回值 (size_t 类型) 的格式说明符

正确选择数据类型对于编写高效且正确的 C 程序至关重要。它影响内存使用、程序性能以及可以存储的数据范围。
